# 🏡 Real Estate Management System

A full real estate platform for managing properties, sales, marketers, commissions, and business analytics.

---

## 🚀 Features

- Property and plot management system
- Sales tracking dashboard
- Marketer/agent commission tracking
- Admin analytics dashboard
- Financial reporting
- PDF export system
- Map-based property tracking

---

## 💡 Problem It Solves

Real estate companies struggle with manual tracking of properties, sales, and commissions.  
This system automates everything in one dashboard.
